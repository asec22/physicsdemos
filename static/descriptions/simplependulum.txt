<p style="text-align: justify;"> We are creating a simple pendulum with a mass at the end of a massless string. Using both linear and non-linear solutions, we solve the second order differential equation <math><mover><mrow><mi>&theta;</mi></mrow><mi>..</mi></mover><mo>=</mo><mfrac><mrow><mo>-</mo><mi>g</mi><mo>sin</mo><mi>&theta;</mi></mrow><mrow><mi>l</mi></mrow></mfrac></math> in state space.  We use state space to set up our system of equations in order to use numerical integration methods like the <a href="https://web.mit.edu/10.001/Web/Course_Notes/Differential_Equations_Notes/node5.html">Runge-Kutta Method</a>. In this simulation, we use the ordinary differential equation libaray from SciPy for the nonlinear solution while the linear solution is <math><mrow><msub><mi>&theta;</mi><mi>o</mi></msub><mo>cos</mo><mo>(</mo><mi>&omega;</mi><mi>t</mi><mo>)</mo></mrow></math>.   </p>
